鸿蒙电脑网页版模拟器，一个Next.js project，基于javascript和ArkTS相似性设计，参考OpenHarmony设计
具备：真实的浏览器功能，开机动画，使用引导，基础功能如开始菜单，任务栏，控制中心，参考了windows和macos两种使用习惯的窗口，设置，文件管理器，记事本，照片查看器等
最大的特点是有别于其他一些电脑模拟网页，具备极致的流畅性丝滑性，拖动窗口，鼠标滑动，调整窗口大小，全都极度跟手，无延迟
鸿蒙电脑模拟器终极版.tar是最初始的压缩包
鸿蒙电脑模拟器终极版-windows本地部署.tar是为了在windows上本地部署，修改了相关linux命令行的压缩包
